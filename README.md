# Python HTTP Parameter Fuzzer

A beginner-friendly web fuzzing and reconnaissance tool built with Python.

## Features

* HTTP parameter fuzzing
* Multi-threaded scanning
* Response analysis
* Status code detection
* Colored terminal output
* Automatic result saving

## Technologies Used

* Python
* Requests
* Threading
* Colorama

## Usage

```bash id="hjoflt"
python fuzzer.py
```
